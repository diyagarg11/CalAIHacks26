# Backend

Single Next.js server. One command, one port.

```
backend/nextjs/          Next.js API server — port 3000
```

See [`nextjs/README.md`](./nextjs/README.md) for full setup and API documentation.

## Quick start

```bash
cd backend/nextjs
npm install
cp .env.local.example .env.local   # fill in keys
npm run dev                         # http://localhost:3000
```
